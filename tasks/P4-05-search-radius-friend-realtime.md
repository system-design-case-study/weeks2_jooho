# P4-05: 검색 반경 표시 + 친구 위치 실시간 갱신

## 메타정보
- Phase: 4
- 의존성: P4-03, P4-01
- 예상 소요: 45분
- 난이도: 중

## 목표
> 자기 위치 중심으로 검색 반경 원이 표시되고, FRIEND_LOCATION 메시지 수신 시 친구 위치가 실시간으로 갱신되며, 반경 내/외에 따라 친구 캐릭터의 색상이 변한다.

## 컨텍스트
> design.md "실시간 위치 공유 + 거리 필터링" 기능의 프론트엔드 구현. 서버에서 FRIEND_LOCATION 메시지를 수신하면 `inRange` 필드로 반경 내/외를 판별하고, Canvas에 반영한다. 검색 반경 슬라이더는 Phase 2 확장(design.md "범위 > Phase 2")에 해당하지만, 기본 반경 표시는 MVP에 포함.

## 상세 요구사항
1. 검색 반경 원 표시
   - 자기 위치 중심으로 원 그리기 (`ctx.arc()`)
   - 반경: INIT 메시지의 `searchRadius` 값 (기본 200)
   - 스타일: 반투명 채움 (`rgba(0, 120, 255, 0.08)`) + 테두리 (`rgba(0, 120, 255, 0.3)`)
   - 렌더링 순서: 배경/그리드 → 반경 원 → 캐릭터
2. FRIEND_LOCATION 수신 시 친구 위치 갱신
   - P4-01의 WebSocket 메시지 분기에서 FRIEND_LOCATION 처리
   - `useRef`의 friends Map에 친구 위치/distance/inRange 갱신
   - 갱신된 데이터가 다음 렌더링 프레임에서 Canvas에 반영
3. 반경 내/외 색상 변경
   - `inRange: true` → 친구 캐릭터 원래 HSL 색상 (P4-03)
   - `inRange: false` → 친구 캐릭터 회색 (P4-03)
   - 색상 전환이 부드럽게 보이도록 (매 프레임 갱신이므로 즉시 반영)
4. 검색 반경 슬라이더 UI (Phase 2 확장)
   - HTML range input 슬라이더
   - 범위: 50 ~ 500 (좌표 단위)
   - 슬라이더 변경 시 반경 원 크기 즉시 갱신
   - 서버에 반경 변경을 알릴 필요는 없음 (프론트엔드 로컬 필터링만)
   - 반경 변경 시 기존 친구의 inRange 상태를 프론트엔드에서 재계산하지 않음 (서버가 기존 반경 기준으로 전송, 프론트엔드는 서버의 inRange를 신뢰)
5. 거리 표시 (선택)
   - 친구 캐릭터 근처에 거리 값 텍스트 표시 (예: "142m")
   - 반경 내 친구에게만 표시

## 설정 참조
```typescript
// 반경 원 렌더링
ctx.beginPath();
ctx.arc(myX, myY, searchRadius, 0, Math.PI * 2);
ctx.fillStyle = 'rgba(0, 120, 255, 0.08)';
ctx.fill();
ctx.strokeStyle = 'rgba(0, 120, 255, 0.3)';
ctx.lineWidth = 1;
ctx.stroke();
```

## 테스트 기준
- [ ] 자기 위치 중심으로 반경 원이 Canvas에 표시되는지 확인
- [ ] FRIEND_LOCATION 수신 시 친구 위치가 Canvas에서 즉시 갱신되는지 확인
- [ ] 반경 내 친구가 원래 색상으로 표시되는지 확인
- [ ] 반경 밖 친구가 회색으로 표시되는지 확인
- [ ] 검색 반경 슬라이더 변경 시 반경 원 크기가 즉시 갱신되는지 확인
- [ ] 두 브라우저 탭에서 하나의 캐릭터를 드래그하면 다른 탭에서 위치가 갱신되는지 확인

## 주의사항
> - design.md: 검색 반경 기본값 200 (INIT 메시지에 포함)
> - design.md: 반경 내 친구는 원래 색상, 반경 밖 친구는 회색 (도트 캐릭터 시스템 섹션)
> - lessons.md: Canvas 데이터(`useRef`)를 갱신하는 것이므로 React 리렌더링을 트리거하지 않음. 슬라이더 UI만 `useState`로 관리
> - design.md: 서버가 `inRange` 필드를 포함하여 전송하므로, 프론트엔드는 자체 거리 계산 불필요 (서버 결과 신뢰)
